# WhatABubble

`WhatABubble` ist eine Mod für Minecraft mit Fabric.
Sie wandelt gesprochene Sprache in sichtbare Sprechblasen um, die über Spielerinnen und Spielern angezeigt werden.

Statt nur zu hören, was gesagt wurde, kann man Gesprochenes also auch direkt im Spiel lesen.
Das ist besonders nützlich, wenn jemand leise spricht, ein Mikrofon schwer zu verstehen ist oder unterschiedliche Sprachen verwendet werden.

## Kurz gesagt

Die Mod kann:

- Gesprochenes erkennen
- daraus Sprechblasen im Spiel erzeugen
- empfangene Sprechblasen auf Wunsch übersetzen
- Farben, Mikrofon und Darstellung anpassen

## Für wen ist das hilfreich?

`WhatABubble` ist zum Beispiel praktisch, wenn:

- in einer Gruppe viele Personen gleichzeitig reden
- man Gesprochenes zusätzlich lesen möchte
- auf einem Server verschiedene Sprachen gesprochen werden
- Sprache im Spiel sichtbarer und verständlicher werden soll
