# WhatABubble Documentation

WhatABubble displays spoken speech as speech bubbles above players. Your own bubble is only visible in **Third Person**, bubbles are hidden while sneaking, and the renderer always uses **Nine-Slice** scaling.

## Documentation

- [`setup.md`](setup.md) – installation, first launch, settings menu, and basic usage
- [`model.md`](model.md) – Vosk model setup, model paths, and speech hints
- [`bubbles.md`](bubbles.md) – bubble visibility, colors, textures, padding, and slice borders
- [`translation.md`](translation.md) – received-bubble translation and local LibreTranslate setup
- [`troubleshooting.md`](troubleshooting.md) – common problems and quick fixes

## Quick Start

1. Install the mod.
2. Copy a Vosk model to `config/whatabubble/vosk-model`.
3. Enter a world.
4. Open the settings menu with `B`.
5. Configure your microphone and bubble settings.
6. Optionally enable translation via LibreTranslate.
